# Natural-Language-Processing (CS60075)

Repository for course assignments and experiments related to Natural Language Processing.

**Structure**
- `A1 - POS Tagging + Figurative LR/`: Assignment 1 notebooks and resources. Key file: `A1-Notebook.ipynb`.
- `A2 - Building Mini CoPilot/`: Assignment 2 notebooks and experiments. Key file: `A2-Notebook.ipynb`.
- `A3 - BERT for Fake Post Classification/`: Assignment 3 notebooks and experiments. Key file: `A3-Notebook.ipynb`.
- `Slides/`: Lecture slides


**Notes by folder**
- `A1 - POS Tagging + Figurative LR`: Contains experiments for part-of-speech tagging and figurative language recognition. Check the first cells for data-download or preprocessing instructions.
- `A2 - Building Mini CoPilot`: Interactive notebooks demonstrating a small code-assistant-style model or pipeline. Review setup cells for model/data requirements.
- `A3 - BERT for Fake Post Classification`: BERT-based experiments for classifying fake posts. These notebooks may require `torch` and `transformers`.
